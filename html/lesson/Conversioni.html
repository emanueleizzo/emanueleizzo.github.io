Sistemi di numerazione

Partiamo dal definire cosa è un sistema di numerazione:
Un sistema di numerazione è un insieme di simboli e di regole con cui rappresentare i numeri ele operazioni che si possono effettuare su di essi.
Per definire un sistema di numerazione quindi è necessario definire due elementi distinti:
  • Un insieme di cifre chiamate simboli.
  • Una sintassi, ossia un insieme di regole che specifica come costruire i vari numeri.
I sistemi di numerazione si possono dividere in due categorie:
  • I sistemi di numerazione posizionale in cui i valori dei simboli variano in base alla posizione che occupano nel numero. Un esempio di sistema di numerazione posizionale è il sistema di numerazione decimale.
  • I sistemi di numerazione additivi/sottrattivo in cui i valori dei simboli non variano e ogni numero è ricavato tramite la somma o la sottrazione delle cifre da cui è formato. Un esempio di sistema di numerazione posizionale è il sistema di numerazione romano.

Il sistema di numerazione decimale

Il sistema di numerazione decimale, o sistema in base 10, è un sistema posizionale che utilizza le cifre decimali da 0 a 9.
Come accennato prima, il sistema di numerazione decimale è un sistema di numerazione posizionale, ossia le cifre utilizzabili variano di valore in base alla posizione in cui si trovano. Questo diventa evidente se si osserva la notazione polinomiale (o espansa) di un qualsiasi numero; esempio:
\[463,23=4\times100+6\times10+3\times1+2\times\frac{1}{10}+3\times\frac{1}{100}=4\times10^2+6\times10^6+3\times10^0+2\times10^{-1}+2\times10^{-3}\]
Generalizzando, è possibile esprimere un numero \(c_{n-1}c_{n-2}...c_1c_0,c_{-1}c_{-2}...\) in base \(b\) come segue:
\[(c_{n-1}c_{n-2}...c_1c_0,c_{-1}c_{-2}...)_b=c_{n-1}\times b^{n-1}+c_{n-2}\times b^{n-2}+...+c_1\times b^1+c_0\times b^0+c_{-1}\times b^{-1}+c_{-2}\times b^{-2}+...\]

Il sistema di numerazione binario

Il sistema di numerazione binario, o sistema in base 2, è un sistema posizionale che utilizza le cifre 0 e 1.
Prima di andare avanti è importante dare la definizione di bit (o binary digit), ossia la singola cifra utilizzata in binario. Come il sistema decimale anche il sistema binario è un sistema posizionale, ossia ad ogni posizione è associata una specifica potenza di 2; esempio:
\[1001011,1101_2=1\times2^6+0\times2^5+0\times2^4+1\times2^3+0\times2^2+1\times2^1+1\times2^0+1\times2^{-1}+1\times2^{-2}+0\times2^{-3}+1\times2^{-4}=64+8+2+1+0.5+0.25+0.0625=75,8125_{10}\]
Notare che utilizzando la notazione polinomiale è possibile passare dalla base 10 alla base, permettendoci quindi di effettuare la conversione da binario a decimale.
