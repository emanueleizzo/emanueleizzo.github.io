INDICE

1) Il programma "Hello world!"

Partiamo dal primo codice di esempio fatto per ogni linguaggio di programmazione, ossia la stampa su schermo della frase "Hello world!"; in Java il programma che esegue ciò è il seguente: 
class Main {
    public statc void main(String[] args) {
        System.out.println("Hello world!");
    }
}

1.1) La definizione della classe

La riga numero 1 contiene la definizione della classe. Come visto prima, la classe è la descrizione effettiva del nostro oggetto, dove un'oggetto è un'entità che ha rilevanza nella risoluzione del nostro problema. Nel nostro esempio è stata creata una classe chiamata Main; un dettaglio importante da ricordare è che il nome della classe deve essere lo stesso di quello del file che la contiene.

1.2) La funzione main

Osserviamo ora la seconda riga del nostro codice: possiamo vedere che viene definita la funzione public static void main(String[] args) { }: questa funzione è il punto di partenza del nostro programma, e deve esistere almeno una classe che la implementi affinché il nostro programma possa girare. Il parametro tra parentesi rappresenta i parametri che possiamo passare al nostro programma esternamente nel momento della sua esecuzione.

1.3) La stampa su schermo

A questo punto manca solo da analizzare la terza riga del codice: in questo caso viene chiamata la funzione System.out.println(), la quale permette di stampare su schermo la stringa passata come parametro richiamando la funzione dalla libreria System.out.
