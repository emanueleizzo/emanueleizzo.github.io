1) Introduzione

Java è un linguaggio di programmazione nato nel 1995 di alto livello, ossia molto più vicino all'utente rispetto che alla macchina, pensato per implementare la programmazione orientata agli oggetti. È un linguaggio portabile, ossia può essere eseguito su piattaforme diverse senza dover essere modificato e ricompilato (si ricorda che con piattaforma si intende l'insieme dell'architettura hardware e software di un elaboratore). Questo perché Java, nel momento in cui un codice viene compilato, genera un bytecode indipendente dalla piattaforma usata, che richiede però di essere eseguito su una macchina virtuale, detta Java Virtual Machine (JVM). Un'altra caratteristica importante di Java è la gestione della memoria, la quale viene effettuata direttamente dal sistema di runtime.

1.1) Il Java Development Kit

Il Java Development Kit (JVM) è un ambiente di sviluppo gratuito privo di interfaccio grafica che permette la compilazione e l'esecuzione di un programma Java tramite riga di comanda. È possibile scaricare il JDK a questo indirizzo.

1.2) Eclipse

Eclipse è un integrated development environment (IDE), ossia un ambiente di sviluppo integrato gratuito che permette la scrittura, la complazione e l'esecuzione di programmi Java. È possibile scaricare Eclipse a questo indirizzo.
