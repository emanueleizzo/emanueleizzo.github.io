1) Gli identificatori e le parole chiave

Prima di parlare delle variabili è necessario aprire una piccola parentesi sugli identificatori e sulle parole chiave. Gli identificatori sono i nomi che il programmatore o la programmatrice assegnano alle specifiche componenti del suo programma, per identificare le variabili, i metodi e le classi, mentre le parole chiave sono un insieme di parole riservate di Java che non possono essere usate come identificatori.

2) Le variabili

Ora che abbiamo introdotto la nozione di identificatore, ossia le variabili. Le variabili sono i contenitori con i quali è possibile memorizzare i dati e possono cambiare valore durante l'esecuzione del programma. Gli elementi che caratterizzano una variabile sono il tipo, il nome e la visibilità (che vedremo successivamente). 
class Main {
    public statc void main(String[] args) {
        // tipo nomevariabile
        int contatore = 5;
        double prezzo = 15.99;
        bool flag = true;
        char lettera = 'A';
        String frase = "Hello world!";
    }
}
Nel nostro esempio sono state definite 5 variabili differenti:
  • Una variabile intera, chiamata "contatore" e con valore pari a 5.
  • Una variabile decimale o in virgola mobile, chiamata "prezzo" e con valore pari a 15.99. Notare che per la parte decimale viene usato il punto (notazione americana) e non la virgola (notazione europea).
  • Una variabile booleana, chiamata "flag" e con valore true. Le variabili booleane possono assumere solo due valori: true (vero) e false (falso).
  • Una variabile alfanumerica contenente un carattere, chiamata "lettera" e contenente la lettera 'A'. Notare che per indicare un singolo carattere si usano gli apici.
  • Una variabile alfanumerica contenente una stringa, chiamata "frase" e contenente la stringa "Hello world!". In questo caso invece la sequenza di caratteri alfenumerici è racchiusa tra virgolette invece che tra apici.

2.1) I tipi di dato

Il tipo di dato stabilisce qual è l'insieme di valori e operazioni accettabili da una variabile. In Java si distinguono due tipi di dato:
  • I tipi primitivi che sono analoghi a quelli offerti da tutti i linguaggi di programmazione. Possono essere raggruppati in tre categorie: numerici, carattere e booleani.
  • I tipi riferimento che sono gli array e le classi, che vedremo nelle prossime lezioni.
